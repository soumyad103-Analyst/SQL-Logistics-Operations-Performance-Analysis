# SQL-Logistics-Operations-Performance-Analysis
Simulated an Operations Analyst role in a logistics company to evaluate delivery efficiency and driver performance.

# Dataset Structure
4 relational tables:
warehouses
drivers
shipments
delivery_status

# Business Objectives
Calculate overall On-Time Delivery %
Compare warehouse-level performance
Rank drivers based on delivery efficiency
Identify high-delay cities

# SQL Concepts Used
Multi-table JOIN
Conditional aggregation using CASE
Percentage calculations
Ranking logic
Operational KPI computation

# Key Insights
On-time delivery rate highlighted operational efficiency gaps
Certain warehouses showed higher delay percentage
Driver experience correlated with performance trends
Specific cities contributed to repeated late deliveries

# Business Recommendations
Investigate warehouse-level bottlenecks

Optimize route allocation for delay-prone cities

Implement driver performance monitoring system
